> # RecoFarm ( Flutter Project )

<br>

> ## 프로젝트 소개
초보 농부를 위한 노지 배추 수확량 & 수익률 예측 App

<br>

> ## 개발 배경
농산물의 가격 변동을 예측하여 농부들에게 안정적인 공급가를 보장

> ## 주요 기능
1. 회원 가입
2. 구글 로그인
3. Open API를 이용한 Chart 구성
4. RandomForest를 이용한 수확량 예측
5. 시계열 데이터 분석 및 ARIMA Model을 이용한 예측
